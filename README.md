# MiniProject

Dengan TailorHub, kami mempersembahkan solusi inovatif untuk memudahkan para pembeli dalam memantau setiap langkah pesanan jahitan mereka tanpa harus meninggalkan kenyamanan rumah. Aplikasi ini memberikan kemudahan akses dan transparansi penuh, menghubungkan admin dan pembeli dalam pengalaman belanja tailor yang tak tertandingi

## Multi-Actor Accessibility:
Dua actor utama: Admin dan Pembeli.

## Fasilitas Admin:

Admin dapat membuat data pesanan dengan cepat dan mudah.
Kemampuan untuk memperbarui status pesanan pembeli secara real-time.
Menentukan dan memperbarui estimasi waktu dengan fleksibilitas.
Kemampuan untuk menghapus data pesanan yang tidak jadi diorder.

## Fasilitas Pembeli:

Pembeli dapat menelusuri daftar baju yang tersedia untuk dipesan.
Proses pemesanan baju yang simpel dan efisien.
Pemantauan status pesanan berdasarkan ID dengan sekali sentuh.
Akses cepat untuk melihat estimasi waktu pengiriman berdasarkan ID pesanan.

### TailorHub memberikan solusi terbaik bagi industri jahitan, menggabungkan kepraktisan administratif dan kenyamanan pelanggan. Jahit lebih mudah dengan TailorHub!"
